# student-performance-predictor
A machine learning-based project designed to predict student academic performance using data-driven insights. This project analyzes various factors such as study time, attendance, previous scores, and family background to forecast student outcomes.

Project Features:

Predicts student performance using ML algorithms

Clean and well-structured data preprocessing pipeline

Exploratory Data Analysis with meaningful visualizations

Comparison of multiple models to select the best performer

Evaluation using standard performance metrics

Easy-to-understand and beginner-friendly code
